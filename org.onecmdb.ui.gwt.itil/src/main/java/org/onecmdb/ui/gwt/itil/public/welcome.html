OneCMDB applications

Here you find applications developed on top of the OneCMDB engine. The applications cover both Service Management and administration and design of OneCMDB systems.

More information about OneCMDB and the applications is available at www.onecmdb.org. There you will find User and Developer Manuals, Tutorials, downloads, legal information and much more. You will also find contact information to Lokomo Systems, which has developed OneCMDB. Lokomo can assist you with commercial licenses, integration work, adaptations, application development etc.
