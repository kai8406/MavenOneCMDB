DATA SOURCE 

Select the type of the data source.

Excel:
  Select an Excel file previously stored in the OneCMDB Repository. 
    Use the upload button to upload an external Excel file to the OneCMDB Repository.  
  Select which worksheet (with its name (i.e. Sheet 1) or number (i.e. 0, 1, 2, etc) 
   you like to import.
  State the Row Number in the sheet that holds the Headings for your data.
  State the Row Number where the data in the sheet begins.
CSV:
  Select a csv file previously stored in the OneCMDB Repository. 
    Use the upload button to upload a csv file to the OneCMDB Repository.  
  State the delimiter that separates each column.
  State the delimiter, if it exists, that encapsulates a column cell. 
  State the Row Number in the sheet that holds the Headings for the data.
  State the Row Number where the data in the sheet begins. 
JDBC (SQL):
  Select a jar library that contains the database jdbc driver. 
    The driver must be located in the OneCMDb Repository.
    Use the upload button to upload a driver to the OneCMDB Repository.
  State the class name of the driver, like com.mysql.jdbc.Driver for mysql.
  State the URL to the database, like mysql:jdbc://localhost:3306/db_name
  State the username/password for authentication.
  State the query to retrieve data.
