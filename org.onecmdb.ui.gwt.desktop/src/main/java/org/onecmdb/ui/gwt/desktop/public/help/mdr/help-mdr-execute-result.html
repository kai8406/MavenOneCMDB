OPEN - EXECUTION RESULT 

The Overview tab presents the result of the transformation, 
i.e. what changes that will be done to the CMDB.
The Details tab presents all these changes in detail, one by one.
If the result looks fine, "click Commit to CMDB", wait for 
the confirmation message and click Close to terminate the Wizard.
If you do not like to import the source data, press cancel and 
either close the Wizard or go back in the Wizard and change the configuration.
