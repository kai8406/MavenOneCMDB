ADD MDR 

An MDR is used for importing data from an external source of data.
For each MDR several different configurations can be created.
This means that different aspects of the data source can be imported.
NOTE: The names can not contain spaces!

Each MDR have its own exceution structure. All MDR related data is
stored under repository/MDR/MDRName and each configuration is 
stored under repository/MDR/MDRName/conf/ConfigName.
