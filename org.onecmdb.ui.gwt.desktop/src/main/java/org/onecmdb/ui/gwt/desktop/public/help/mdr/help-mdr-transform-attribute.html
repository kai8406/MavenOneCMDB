TEMPLATE/ATTRIBUTE MAPPING 

Each data source column can be mapped to a CMDB Template and an Attribute.
Choose a suitable template by clicking in the CMDB Template cell. Then select
which attribute to populate by clicking in the CMDB Attribute cell. 

Each row in the data source will create new CI instances unless restrictions 
on Natural Keys are applied. This is controlled in the Natural Key tab. 

Each row also defines references between the different CMDB templates 
used in the transform. This is updated in the References tab.
